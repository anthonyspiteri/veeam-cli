# Skill Conventions

Canonical conventions for `bakufu-cli` skills.

## Scope

Applies to all generated skill folders under `skills/`:
- Service skills (`bakufu-*`)
- Helper skills (`bakufu-*`)
- Recipe skills (`recipe-*`)
- Persona skills (`persona-*`)

## Source of Truth

Generated by:
- `scripts/sync_skills_from_swagger.py`

Do not hand-edit generated `skills/*/SKILL.md` files unless you also update the generator.

## Required Files Per Skill

Each skill folder must contain:
- `SKILL.md`
- `agents/openai.yaml`

## `SKILL.md` Required Frontmatter

Every `SKILL.md` starts with YAML frontmatter:

```yaml
---
name: "<skill-name>"
version: "1.0.0"
description: "<short description>"
metadata:
  openclaw:
    category: "<service|helper|recipe|persona>"
    requires:
      bins:
        - "bakufu"
      skills:
        - "<required-skill-1>"
        - "<required-skill-2>"
---
```

## Body Structure

### Persona skills

Must include:
- Title (human-readable persona name)
- `PREREQUISITE` line listing required utility skills
- One-line description
- `## Relevant Workflows`
- `## Primary Focus Areas`
- `## Instructions`
- `## Recommended Recipes`
- `## Tips`
- `## Mission`

### Service skills

Must include:
- Title
- `PREREQUISITE` line
- One-line service description
- `## Relevant Commands`
- `## Instructions`
- `## Tips`

### Helper skills

Must include:
- Title
- `PREREQUISITE` line
- One-line helper description
- `## Relevant Commands`
- `## Instructions`
- `## Tips`

### Recipe skills

Must include:
- Title
- `PREREQUISITE` line
- One-line recipe description
- `## Relevant Commands`
- `## Instructions`
- `## Tips`

## `agents/openai.yaml` Required Fields

Every skill must include:

```yaml
display_name: "<Human Friendly Name>"
short_description: "<1-line summary>"
default_prompt: "<default invocation prompt>"
```

## Naming Rules

- Service/helper names: `bakufu-*`
- Recipe names: `recipe-*`
- Persona names: `persona-*`
- Use lowercase + hyphenated names only.

## Contributor Workflow

1. Update `scripts/sync_skills_from_swagger.py`.
2. Regenerate:
   - `python scripts/sync_skills_from_swagger.py`
3. Validate sample outputs:
   - `skills/<name>/SKILL.md`
   - `skills/<name>/agents/openai.yaml`
4. Verify skill index:
   - `docs/skills.md`
   - `bakufu skills list`

## Review Checklist

- Frontmatter present and valid YAML.
- `metadata.openclaw.category` matches skill type.
- `metadata.openclaw.requires.bins` includes `bakufu`.
- `metadata.openclaw.requires.skills` is populated.
- Body sections match category conventions.
- `agents/openai.yaml` exists with required keys.
- No absolute local filesystem paths in generated content.
